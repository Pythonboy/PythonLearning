# 梯度下降法
- 不是一个机器学习算法，而是一种基于搜索的最优化方法
- 作用：最小化一个损失函数
- 梯度上升法： 最大化一个效用函数

**注意** ：并不是所有函数都有唯一的极值点

**解决方案**：
- 多次运行，随机化初始点
- 梯度下降法的初始点也是一个超参数
